# Java CRUD Console Application

## Description
A simple console-based User Management System built using Java.
Implements basic CRUD operations (Create, Read, Delete).

## Features
- Add user
- View users
- Delete user
- Menu-driven interface

## Concepts Used
- OOPS (Encapsulation)
- ArrayList
- Scanner
- Modular programming

## How to Run

1. Compile files:
   javac Main.java User.java

2. Run:
   java Main
